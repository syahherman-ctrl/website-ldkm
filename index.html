How do we do it?

1

Upload Your Files

Your file needs to be temporarily uploaded to our server, to check image data via Hive API and also Synth ID.

2

Synth ID Verification

We share your image with Google Gemini API to check for watermarks, which are applied if the photo was created by Nanobanana.

3

Machine Lerning API

If the image does not have a synth ID watermark, we will check using Hive's AI-Generated & Deepfake Content Detection APIs

4

Conclution

We will provide a conclusion from what we found through Hive API and checking the watermark from Nanobanana.
